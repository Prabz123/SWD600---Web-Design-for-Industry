Footer

  • Working example

    Use the menu below to see an example on a seperate page.

    Example

  • What is it?

    A footer section would be used to provide extra information that may be useful to the user. Certain external links may be shown to additional pages that may give additional information or other pages that haven't been mentioned in the main navigation.

  • HTML Source Code

    Below is the HTML source code for the Footer. The code uses external font awesome stylesheet which would display the social media icons on screen.

    <html lang="en"> <head> <meta charset="UTF-8"> <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ ajax/libs/font-awesome/4.7.0/css/font-awesome.min.css"> </head> <body> <footer class="footer"> <div class="container"> <div class="row"> <div class="col-sm-12 text-center"> <div class="navigation"> <h3>Find Us On Social Media!</h3> </div> <div class="social-icon"> <a href=""><i class="fa fa-facebook"></i></a> <a href=""><i class="fa fa-twitter"></i></a> <a href=""><i class="fa fa-google-plus"></i></a> <a href=""><i class="fa fa-instagram"></i></a> <a href=""><i class="fa fa-youtube"></i></a> </div> <div class="copyright"> <p> &copy; <b>Southampton Solent University</b> </p> <div class="bordernow"></div> </div> </div> </div> </div> </footer> </body> </html>

  • CSS Source Code

    Below is the CSS source code for the Footer. The background has a background image and the icons have border styling and is presented with different border colours. In addition the icon border will become larger when the user hovers over them.

    /* Properties of the footer section */ footer { background-image: url(yellow.jpg); background-size: cover; background-attachment: fixed; padding-top: 50px; padding-bottom: 10px; text-align: center; -webkit-background-size: cover; -moz-background-size: cover; } /* Defining the social media icons */ .social-icon a { margin: 0 16px 20px 16px; display: inline-block; } /* Setting the properties for the social media icons */ .social-icon .fa { font-size: 30px; color: white; width: 70px; height: 60px; border-radius: 30%; background: grey; line-height: 2; transition: .3s; border: 10px solid color #b1c9d1; } /* Establishing the Facebook icon */ .social-icon .fa-facebook { font-size: 30px; color: darkblue; width: 70px; height: 60px; background: white; border: 2px solid blue } /* Establishing the Twitter icon */ .social-icon .fa-twitter { font-size: 30px; color: mediumturquoise; width: 70px; height: 60px; background: white; border: 2px solid mediumturquoise } /* Establishing the Google Plus icon */ .social-icon .fa-google-plus { font-size: 30px; color: red; width: 70px; height: 60px; background: white; border: 2px solid black } /* Establishing the Instagram icon */ .social-icon .fa-instagram { font-size: 30px; color: purple; width: 70px; height: 60px; background: white; border: 2px solid purple; } /* Establishing the YouTube icon */ .social-icon .fa-youtube { font-size: 30px; color: red; width: 70px; height: 60px; background: white; border: 2px solid red; } /* Makes social media icons hover with mouse over */ .social-icon a:hover .fa { border: 5px solid black; } /* The border styles presented around each icon */ .bordernow { width: 80px; border-bottom: 2px solid white; margin: 0 auto; margin-top: 20px; } /* Copyright text */ .copyright { text-align: center; }

Component Usage

The footer section will be placed at the very bottom of the page. It will contain technical information. For example information like copyright information, name of the website's author, business contact details, social media links etc. This component of a website isn't typically relied on too much by the user but still helps in providing external links to find out more information
