  • What is it?

    Navigation bar as it's name suggests is quite simply an element which allows the user to navigate around a website. The navigation stands as one of the more important features of a website as without it, the user would not be able to navigate.

  • HTML Source Code

    Below is the HTML source code for the menu bar. The working example is similar to the navigation bar of this Pattern Library. It is a simple navigation bar focused for the horizontal approach.

    <html lang="en"> <head> </head> <body> <header> <div class="container"> <nav> <ul> <li class="current"><a href="#">Home</a></li> <li><a href="#">Page 1</a></li> <li><a href="#">Page 2</a></li> <li><a href="#">Page 3</a></li> </ul> </nav> </div> </header> </body> </html>

  • CSS Source Code

    Below is the CSS source code for the menu bar. The CSS styling copies exactly that of the main navigation bar of this Pattern Library. With any modern navigation bar, the CSS code shows features for hovering over menu tabs.

    /* Header Section */ header { background: #35424a; color: #ffffff; padding-top: 30px; min-height: 70px; border-bottom: #e8491d 3px solid; } /* Navigation Links */ header a { color: #ffffff; text-decoration: none; text-transform: uppercase; font-size: 16px; } /* Navigation List Items */ header li { float: left; display: inline; padding: 0 20px 0 20px; } /* Header Navigation Bar */ header nav { float: right; margin-top: 10px; } /* Highlight Current Page/And/Or Text */ header .highlight, header .current a { color: #e8491d; font-weight: bold; } /* Hover Over Page Link */ header a:hover { color: #ff8000; font-weight: bold; }

Component Usage

Navigation bars are commonly found on either the top of the page or vertically on either the left or the right side of the page, it allows the user to navigate around the site and view different pages.


Horizontal Menu Bar

Horizontal navigation bars are the more preferred approach when it comes to adding navigation on a website. Horizontal navigation allows the user to clearly see all the links, furthermore it is effective because people are already accustomed to reading from left to right of the screen.


Vertical Menu Bar

Vertical navigation bars are not that common but still used in most sites. These menus are often incorporated into graphic intensive websites such as portfolio, restaurant, salon, spa etc. Using a vertical menu bar is greatly used if there are going to be a lot of menu links and tabs to display.
