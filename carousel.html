Carousel

  • Working example

    As shown above, this example is not a manuel slideshow where the user can click arrows to navigate to the next image but instead the carousel is automatic using keyframes.

  • What is it?

    Carousels can be found on the home page of a particular website. They are mainly used to present information or images. they may be found on the home page to provide basic context for the website.

  • HTML Source Code

    Below is the HTML source code for the Carousel. The working example is automatic and requires no interaction between the user. This animation has been done using keyframes established in the CSS.

    <html lang="en"> <head> <meta charset="UTF-8"> </head> <body> <h1>Slider</h1> <div class="con"> <img src="1.jpg" class="img"> <img src="2.jpg" class="img"> <img src="3.jpg" class="img"> </div> </body> </html>

  • CSS Source Code

    Below is the CSS source code for the Carousel. Reaching the end of the script, you will see keyframes, keyframes enables the carousel to be automatic.

    /* Border and Body of Carousel */ .con { width: 512px; height: 256px; border: 5px solid red; margin: 50px auto; overflow: hidden; position: relative; } /* Placing and sorting out Images in Carousel */ .img { position: absolute; width: 512px; height: 256px; animation: mm 15s infinite; -webkit-animation: mm 15s infinite; opacity: 0; } /* Animation and transition between Images */ @keyframes mm { 10%{opacity:1} 50%{opacity:0} } /* Enabling animation to work on different Browsers */ @-webkit-keyframes mm { 10%{opacity:1} 50%{opacity:0} } /* Sorting out and animating Images in order of display using "nth-child" */ img:nth-child(0){animation-delay: 0s; -webkit-animation-delay: 0s; } img:nth-child(1){animation-delay: 5s; -webkit-animation-delay: 5s;} img:nth-child(2){animation-delay: 10s; -webkit-animation-delay: 10s;}

Component Usage

Image carousels aren't widely used in many websites. They usually house and showcase photos/images, advertise a company's products or simply capture the attention of users using your website.
