Jumbotron

  • Working example

    Use the menu below to see an example on a seperate page.

    Example

  • What is it?

    Jumbotron are basic components that doesn't really have any productive value other than the fact it is specifically used to call the users attention to certain content on your website.

  • HTML Source Code

    Below is the HTML source code for the Jumbotron. The code uses external JQuery and Bootstrap scripts. Bootstrap would be responsible for any icons or buttons styling from the example.

    <html lang="en"> <head> <meta charset="utf-8"> <meta name="viewport" content="width=device-width, initial-scale=1"> <meta http-equiv="x-ua-compatible" content="ie=edge"> <link rel="stylesheet" href="https://cdn.rawgit.com/twbs/bootstrap/ v4-dev/dist/css/bootstrap.css"> </head> <body> <div class="container"> <div class="row"> <div class="col-xs-12 col-sm-6 col-md-4 col-lg-3 col-xl-2 red"> Navigation 1 </div> <div class="col-xs-12 col-sm-6 col-md-4 col-lg-3 col-xl-2 green"> Navigation 2 </div> <div class="col-xs-12 col-sm-12 col-md-12 col-lg-12 col-xl-12"> <br /> <div class="jumbotron"> <h1 class="display-3">Welcome to the Pattern Library!</h1> <p class="lead"> This Library has been created for BSc - Web Design and Development at Southampton Solent University! </p> <hr class="m-y-md" /> <p> This is an example of a Jumbotron.z </p> <p class="lead"> <a class="btn btn-primary btn-lg" href="#" role="button">Example Button</a> </p> </div> </div> </div> </div> <script src="https://ajax.googleapis.com/ajax/libs/ jquery/2.1.4/jquery.min.js"></script> <script src="https://cdn.rawgit.com/twbs/bootstrap/ v4-dev/dist/js/bootstrap.js"></script> </body> </html>

  • CSS Source Code

    Below is the CSS source code for the Jumbotron. Because the script already uses external JQuery and Bootstrap scripts, the only components that need styling is the navigation 1 and navigation 2 from the column components.

    /* Styling for the 1st column */ .red { color: white; background-color: red; } /* Styling for the 2nd column */ .green { color: white; background-color: green; }

Component Usage

Jumbotron is a specific component from the Bootstrap library. The usage for the jumbotron is extending the viewport of the page to showcase key content and information displayed on your site.
